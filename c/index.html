Giới thiệu ngôn ngữ C & Công cụ

Tại sao chọn C?

  • Nền tảng vững chắc: C là ngôn ngữ mẹ của nhiều ngôn ngữ hiện đại (C++, Java, C#, PHP).
  • Hiệu suất cao: Chạy nhanh, tương tác trực tiếp với bộ nhớ và phần cứng.
  • Tính di động: Mã C có thể biên dịch trên nhiều hệ điều hành.

Công cụ lập trình (DevC++)

  • Môi trường phát triển tích hợp (IDE) nhẹ và miễn phí.
  • Tích hợp sẵn trình biên dịch MinGW (GCC).
  • Rất dễ cài đặt và sử dụng cho người mới tiếp cận lập trình.

Chương trình C đầu tiên

/* Chương trình in ra câu chào */
#include <stdio.h>

int main() {
    // Lệnh in nội dung ra màn hình
    printf("Hello, World!\n");
    
    return 0;
}

Giải thích từng dòng lệnh

  • #include <stdio.h>: Khai báo thư viện Standard Input/Output để dùng hàm printf.
  • int main(): Hàm chính. Mọi chương trình C đều bắt đầu chạy từ đây.
  • { }: Cặp ngoặc nhọn bao bọc khối lệnh.
  • printf(...): Hàm xuất dữ liệu. \n dùng để xuống dòng.
  • return 0;: Kết thúc hàm main, báo hiệu chạy thành công.
  • Dấu ;: Lệnh trong C bắt buộc kết thúc bằng chấm phẩy.

Kiểu dữ liệu, Biến và Hằng

Kiểu Dữ Liệu

Xác định loại và kích thước lưu trữ.

  • int: Số nguyên (VD: 10, -5)
  • float: Số thực (VD: 3.14)
  • double: Số thực cực lớn
  • char: Ký tự (VD: 'A')

Biến (Variables)

Vùng nhớ dùng để lưu trữ dữ liệu, có thể thay đổi giá trị.

Cú pháp:
type name = value;

Quy tắc đặt tên:
Không bắt đầu bằng số, không khoảng trắng, phân biệt hoa/thường.

Hằng (Constants)

Giá trị cố định, KHÔNG thể thay đổi khi chạy.

Từ khóa const:
const float PI = 3.14;

Macro:
#define PI 3.14

Các Phép Toán: 1. Phép toán số học

Sử dụng để thực hiện các phép tính toán học cơ bản.

Ký hiệu Ý nghĩa
+Cộng hai toán hạng
-Trừ hai toán hạng
*Nhân hai toán hạng
/Chia hai toán hạng (lấy nguyên nếu là int)
%Chia lấy phần dư (chỉ dùng cho số nguyên)
 
int a = 10, b = 3;

int tong = a + b;   // 13
int hieu = a - b;   // 7
int tich = a * b;   // 30
int thuong = a / b; // 3 (Mất phần thập phân)
int du = a % b;     // 1 (10 chia 3 dư 1)

Các Phép Toán: 2. Phép toán so sánh

Dùng để so sánh 2 giá trị. Kết quả trả về là 1 (Đúng - True) hoặc 0 (Sai - False).

Ký hiệu Ý nghĩa
==Bằng nhau
!=Khác nhau
>Lớn hơn
<Nhỏ hơn
>=Lớn hơn hoặc bằng
<=Nhỏ hơn hoặc bằng
 
int a = 5, b = 8;

printf("%d", a == b); // In ra 0 (Sai)
printf("%d", a != b); // In ra 1 (Đúng)
printf("%d", a < b);  // In ra 1 (Đúng)

// Rất dễ nhầm lẫn giữa = (gán) và == (so sánh)

Dùng để kết hợp nhiều điều kiện với nhau (thường dùng trong lệnh if, while).

Các Phép Toán: 4. Gán và Tăng/Giảm

Phép Gán (Assignment)

Dùng để gán giá trị từ bên phải sang biến bên trái.

=a = 5
+=a += 3 (Tương đương a = a + 3)
-=a -= 2 (Tương đương a = a - 2)
*=, /=Hoạt động tương tự

Phép Tăng / Giảm

Dùng để tăng hoặc giảm biến đi 1 đơn vị.

  • ++a hoặc a++ : Tăng a lên 1.
  • --a hoặc a-- : Giảm a đi 1.
int x = 10;
x++; // Lúc này x = 11
x--; // Trở lại x = 10

Lệnh Nhập/Xuất & Chú Thích (Comment)

#include <stdio.h>

int main() {
    int age;
    float weight;

    // printf dùng để xuất văn bản
    printf("Nhap tuoi cua ban: ");

    /* scanf dùng để đọc dữ liệu từ bàn phím.
       CHÚ Ý: Phải có dấu & trước tên biến */
    scanf("%d", &age);

    printf("Nhap can nang (kg): ");
    scanf("%f", &weight);

    // In dữ liệu ra màn hình
    printf("Ban %d tuoi, %.1f kg.", age, weight);
    return 0;
}

Lưu ý quan trọng

  • Hàm scanf bắt buộc phải có dấu & (ampersand) trước tên biến (VD: &age) để truyền địa chỉ bộ nhớ.
  • Hàm printf KHÔNG CẦN dấu & khi in giá trị ra.
  • Chú thích 1 dòng: Dùng // nội dung
  • Chú thích nhiều dòng: Dùng /* nội dung */
  • Máy tính sẽ bỏ qua chú thích khi chạy chương trình. Nhằm mục đích cho người đọc code hiểu.

Chuỗi định dạng (Format Specifiers)

Chuỗi định dạng được sử dụng trong printf và scanf để cho máy tính biết bạn đang muốn làm việc với loại dữ liệu nào.

Ký hiệu Kiểu dữ liệu Giải thích & Ví dụ minh họa
%d int (Số nguyên) Dùng cho số nguyên. VD: printf("Tuổi: %d", 20);
%f float (Số thực) Số thực độ chính xác thường. Có thể dùng %.2f để làm tròn 2 chữ số thập phân.
%lf double (Số thực lớn) Dùng khi cần số thực chính xác cao. Bắt buộc dùng %lf khi scanf kiểu double.
%c char (Ký tự) In ra 1 ký tự duy nhất. VD: printf("Hạng: %c", 'A');
%s char[] (Chuỗi) In ra chuỗi ký tự (mảng char). VD: printf("Tên: %s", "John");

Bài tập vận dụng: Chương 1

  • Bài 1: Viết chương trình nhập vào 2 số nguyên a và b. Thực hiện hoán vị (đổi chỗ) giá trị của 2 biến này và in kết quả ra màn hình.
  • Bài 2: Nhập vào bán kính r của một hình tròn. Tính và in ra chu vi và diện tích của hình tròn đó (Sử dụng const float PI = 3.14;).
  • Bài 3: Viết chương trình tính diện tích và chu vi hình chữ nhật với chiều dài và chiều rộng được nhập từ bàn phím.
  • Bài 4: Nhập vào một số nguyên x. Tính và in ra giá trị bình phương của x.
  • Bài 5: Nhập vào một số thực dương n. Hãy tính và in ra căn bậc 2 của n. (Gợi ý: Sử dụng thư viện #include <math.h> và hàm sqrt(n)).

Cấu Trúc Điều Khiển

Lệnh rẽ nhánh (If, Switch) và Vòng lặp (For, While)

Cấu trúc rẽ nhánh (If - Else)

1. Lệnh if cơ bản

Chỉ thực hiện nếu điều kiện ĐÚNG.

if (tuoi >= 18) {
    printf("Duoc phep lai xe.");
}

2. Lệnh if - else

Xử lý 2 trường hợp Đúng / Sai.

if (diem >= 5.0) {
    printf("Dau");
} else {
    printf("Rot");
}

3. Lệnh if - else if - else

Kiểm tra nhiều điều kiện liên tiếp nhau.

float diem = 7.5;

if (diem >= 8.0) {
    printf("Xep loai: Gioi");
} 
else if (diem >= 6.5) {
    printf("Xep loai: Kha");
} 
else if (diem >= 5.0) {
    printf("Xep loai: Trung Binh");
} 
else {
    // Chạy khi tất cả các đk trên đều SAI
    printf("Xep loai: Yeu");
}

Câu lệnh Switch - Case

Sử dụng thay thế cho if-else if kéo dài khi biến chỉ nhận các giá trị rời rạc cụ thể (số nguyên int hoặc ký tự char).

  • case: Xác định giá trị cụ thể.
  • break: Thoát khỏi switch, nếu thiếu sẽ bị "trôi" sang case tiếp theo.
  • default: Chạy khi không khớp case nào (tương đương else).
int lua_chon = 2;

switch (lua_chon) {
    case 1:
        printf("Ban chon Them moi");
        break;
    case 2:
        printf("Ban chon Chinh sua");
        break;
    case 3:
        printf("Ban chon Xoa");
        break;
    default:
        printf("Lua chon khong hop le!");
}

Bài tập vận dụng: If & Switch

  • Bài 1: Nhập vào một số nguyên n. Kiểm tra và thông báo n là số chẵn hay số lẻ ra màn hình.
  • Bài 2: Nhập vào 3 số thực a, b, c. Tìm và in ra giá trị lớn nhất trong 3 số đó.
  • Bài 3: Giải phương trình bậc nhất: ax + b = 0. Hãy xét kỹ trường hợp a=0 (phương trình vô nghiệm hoặc vô số nghiệm) và a khác 0.
  • Bài 4: Tính tiền điện: Nhập số chữ điện tiêu thụ. Tính theo giá lũy tiến: 50 chữ đầu (1500đ/chữ), 50 chữ tiếp theo (2000đ/chữ), phần còn lại (3000đ/chữ).
  • Bài 5: Viết chương trình máy tính cầm tay. Nhập 2 số và 1 ký tự phép toán (+, -, *, /). Dùng switch-case để kiểm tra phép toán, in ra kết quả.

Vòng lặp For

Định nghĩa và Cú pháp

Sử dụng khi bạn đã biết trước số lần lặp cụ thể.

for (khởi_tạo; điều_kiện; bước_nhảy) {
    // Khối lệnh lặp lại
}
  • Khởi tạo: Thực hiện 1 lần duy nhất đầu tiên (VD: int i = 1;).
  • Điều kiện: Kiểm tra trước mỗi vòng lặp. Đúng thì chạy, Sai thì dừng (VD: i <= 10;).
  • Bước nhảy: Thực hiện vào cuối mỗi vòng (VD: i++).
// Ví dụ: In ra các số từ 1 đến 5
int i;
for (i = 1; i <= 5; i++) {
    printf("%d ", i);
}
// Kết quả in ra: 1 2 3 4 5

// Vòng lặp chạy lùi
for (i = 5; i > 0; i--) {
    printf("%d ", i);
}
// Kết quả: 5 4 3 2 1

Bài tập vận dụng: Vòng lặp For

  • Bài 1: Nhập vào số nguyên dương n. Viết vòng lặp tính tổng: S = 1 + 2 + 3 + ... + n.
  • Bài 2: Nhập vào số nguyên dương n, tính giai thừa của n (n! = 1 * 2 * ... * n).
  • Bài 3: In ra màn hình bảng cửu chương của một số N (Nhập N từ 2 đến 9).
  • Bài 4: Nhập vào số nguyên dương n. Tìm và in ra tất cả các ước số của n.
  • Bài 5: Vẽ hình chữ nhật đặc bằng các dấu * dựa vào chiều dài và chiều rộng nhập từ bàn phím (sử dụng 2 vòng lặp lồng nhau).

Vòng lặp While & Do-While

Vòng lặp while

Kiểm tra điều kiện TRƯỚC. Dùng khi chưa biết trước số lần lặp.

int n = 5;
while (n > 0) {
    printf("%d ", n);
    n--; // Quan trọng: tránh lặp vô hạn
}

Vòng lặp do - while

Thực hiện lệnh ÍT NHẤT 1 LẦN, sau đó mới kiểm tra điều kiện. (Thường dùng ép người dùng nhập lại).

int age;
do {
    printf("Nhap tuoi (>0): ");
    scanf("%d", &age);
} while (age <= 0);

Lưu ý: Cẩn thận với vòng lặp vô tận (Infinite Loops) nếu quên thay đổi giá trị của điều kiện dừng.

Bài tập vận dụng: While & Do-While

  • Bài 1: Dùng do-while để yêu cầu nhập 1 số trong khoảng 1 đến 10. Nếu nhập sai, in thông báo lỗi và yêu cầu nhập lại liên tục.
  • Bài 2: Nhập vào một số nguyên dương. Dùng while để đếm xem số đó có bao nhiêu chữ số (bằng cách chia liên tiếp cho 10).
  • Bài 3: Nhập vào một số nguyên dương n. Hãy đảo ngược thứ tự các chữ số của n và in ra.
  • Bài 4: Tìm Ước chung lớn nhất (UCLN) của hai số a và b bằng thuật toán Euclid (chia lấy dư trong vòng lặp while).
  • Bài 5: Trò chơi đoán số: Random 1 số (1-100). Cho người chơi nhập vòng lặp, in ra gợi ý "Lớn hơn" hoặc "Nhỏ hơn" cho tới khi đoán trúng.

Hàm & Cấu Trúc Chương Trình

Tái sử dụng mã nguồn và chia nhỏ chương trình

Hàm có giá trị trả về (Return)

Hàm là một khối lệnh độc lập nhận đầu vào (tham số), xử lý và trả về một kết quả ra bên ngoài thông qua từ khóa return.

  • Kiểu trả về: Là int, float, char... ứng với giá trị trả ra.
  • return: Ngay khi gặp lệnh return, hàm sẽ lập tức kết thúc.
// Hàm nhận vào 2 số, trả về 1 số int
int tinhTong(int a, int b) {
    int ketQua = a + b;
    return ketQua; // Trả giá trị ra
}

int main() {
    // Gọi hàm và lưu kết quả vào biến
    int tongToanLy = tinhTong(8, 9);
    printf("Tong la: %d", tongToanLy);
    return 0;
}

Hàm KHÔNG trả về giá trị (Void)

Được sử dụng khi hàm chỉ thực hiện các thao tác (như in ra màn hình, ghi file) mà không cần xuất ra giá trị nào để tính toán tiếp.

  • Sử dụng từ khóa void thay cho kiểu dữ liệu.
  • Không bắt buộc phải có lệnh return; ở cuối hàm.
// Hàm void chỉ in ra, không trả về gì
void inLoiChao(char ten[]) {
    printf("==================\n");
    printf("Xin chao ban %s!\n", ten);
    printf("==================\n");
}

int main() {
    // Chỉ cần gọi hàm là chạy lệnh bên trong
    inLoiChao("Sinh vien");
    return 0;
}

Bài tập vận dụng: Hàm

  • Bài 1: Viết hàm int kiemTraSNT(int n) trả về 1 nếu n là số nguyên tố, trả về 0 nếu sai. Dùng hàm này in các SNT từ 1 đến 100.
  • Bài 2: Viết hàm int timMax(int a, int b, int c) trả về số lớn nhất.
  • Bài 3: Viết hàm float chuyenDoiDoC(float doF) chuyển đổi từ Fahrenheit sang Celsius: C = (F - 32) * 5/9.
  • Bài 4: Viết hàm void veHinhVuong(int canh) chỉ làm nhiệm vụ in ra màn hình một hình vuông dấu * cạnh bằng N.
  • Bài 5: Viết hàm int tinhFibonacci(int n) trả về số Fibonacci thứ n trong dãy (F0=0, F1=1, Fn = Fn-1 + Fn-2).

Làm việc với nhiều File Header (.h)

Chia nhỏ chương trình thành các module để quản lý dự án lớn dễ dàng hơn.

1. File Header (math_utils.h)

Chứa "Khai báo hàm" (Declaration).

#ifndef MATH_UTILS_H

int cong(int a, int b);

#endif

2. File Mã Nguồn (main.c)

Include file .h (bằng dấu " ") để sử dụng.

#include <stdio.h>

    printf("KQ = %d", kq);

  • Bài 2: Tạo thư viện geometry.h định nghĩa các hàm tính diện tích/chu vi hình tròn, hình chữ nhật. Viết main.c để dùng.
  • Bài 3: Viết một thư viện string_utils.h chứa hàm để đếm độ dài của một chuỗi ký tự tự viết.
  • Bài 4: Phân tách hàm kiểm tra nguyên tố (SNT) ra một file number_check.h riêng biệt và tích hợp vào main.c.

Cấu Trúc Dữ Liệu Nâng Cao

Mảng, Struct, Con Trỏ & Danh Sách Liên Kết

Mảng 1 chiều (1D Arrays)

Khái niệm

Tập hợp các biến có cùng kiểu dữ liệu, được lưu trữ ở các ô nhớ liên tiếp nhau.

  • Chỉ số (Index): Bắt đầu từ 0 đến n-1.
  • Khai báo: int arr[5];
  • Khởi tạo: int arr[3] = {10, 20, 30};
// Duyệt mảng bằng vòng lặp for
int diem[4] = {7, 8, 9, 10};

for (int i = 0; i < 4; i++) {
    printf("Diem phan tu %d la: %d\n", i, diem[i]);
}

Bài tập vận dụng:
1. Khai báo mảng N phần tử. Viết vòng lặp nhập xuất mảng, tính tổng.
2. Tìm phần tử lớn nhất (MAX) và nhỏ nhất (MIN) trong mảng.
3. Viết thuật toán Bubble Sort sắp xếp mảng tăng dần.

Struct (Kiểu cấu trúc)

Dùng để nhóm nhiều loại dữ liệu khác nhau (int, float, char...) thuộc về một đối tượng cụ thể (VD: Sinh viên, Tọa độ).

Sử dụng dấu chấm . để truy cập thuộc tính.

struct SinhVien {
    int id;
    float diem;
};

int main() {
    // Khai báo biến kiểu struct
    struct SinhVien sv1;
    
    // Gán giá trị
    sv1.id = 101; 
    sv1.diem = 8.5;

    printf("ID: %d, Diem: %.1f", sv1.id, sv1.diem);
    return 0;
}

Bài tập vận dụng:
1. Struct ToaDo (x, y). Viết hàm tính khoảng cách 2 điểm.
2. Struct NhanVien. Tạo mảng 5 NV, in ra NV lương cao nhất.

Con Trỏ (Pointers) - Trái tim của C

Con trỏ là một biến đặc biệt. Thay vì chứa một giá trị (như 5, 10), nó chứa địa chỉ bộ nhớ của biến khác.

  • &a : Lấy địa chỉ của biến a.
  • *p : Khai báo con trỏ HOẶC lấy giá trị tại địa chỉ mà con trỏ đang trỏ tới.
int a = 10;
int *p = &a; // p trỏ tới biến a

printf("Gia tri cua a: %d\n", a);
printf("Dia chi cua a: %p\n", &a);
printf("Gia tri ma p tro toi: %d\n", *p);

*p = 20; // Thay đổi giá trị tại địa chỉ đó
printf("a bay gio la: %d", a); // in ra 20

Danh sách liên kết (Linked List) - Khái niệm

Khác với mảng (kích thước cố định, ô nhớ liên tiếp), danh sách liên kết là cấu trúc động. Các khối dữ liệu (gọi là Node) nằm rải rác trong bộ nhớ và móc nối với nhau qua con trỏ.

[Head] -> [ Data: 10 | Next ] -> [ Data: 20 | Next ] -> NULL

Thành phần 1 Node:

  • Data: Lưu trữ dữ liệu.
  • Next: Con trỏ lưu địa chỉ của Node tiếp theo.
struct Node {
    int data;
    struct Node *next; // Con trỏ trỏ tới Node khác
};

Danh sách liên kết - Cài đặt bằng tay

Đây là cách thức sơ khai nhất để liên kết 3 Node lại với nhau mà không cần dùng hàm cấp phát động phức tạp.

Bài tập vận dụng nâng cao:
1. Viết hàm createNode(int x) sử dụng lệnh malloc để cấp phát động.
2. Viết vòng lặp duyệt qua danh sách liên kết cho đến khi current != NULL để in ra các giá trị.
3. Viết hàm thêm 1 Node vào đầu danh sách (insertAtHead).

int main() {
    // Tạo 3 node tĩnh
    struct Node n1, n2, n3;

    // Gán dữ liệu
    n1.data = 10;
    n2.data = 20;
    n3.data = 30;

    // Liên kết chúng lại bằng con trỏ next
    n1.next = &n2;
    n2.next = &n3;
    n3.next = NULL; // Kết thúc danh sách

    // In dữ liệu thông qua con trỏ
    printf("Node 2 la: %d\n", n1.next->data);

    return 0;
}
