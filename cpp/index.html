Giới thiệu ngôn ngữ C++ và Công cụ

Tại sao chọn C++?

  • Kết hợp sức mạnh: Hỗ trợ lập trình thủ tục, hướng đối tượng và generic.
  • Hiệu suất cao: Phù hợp game, hệ thống nhúng, ứng dụng hiệu năng cao.
  • Hệ sinh thái lớn: STL, thư viện đa dạng, sử dụng rộng rãi trong công nghiệp.

Công cụ lập trình

  • Visual Studio Code + MinGW/GCC hoặc MSVC.
  • Code::Blocks, Visual Studio Community cho người mới.
  • Biên dịch với tiêu chuẩn C++17/C++20 để dùng tính năng hiện đại.

Trình biên dịch

GCC, Clang, MSVC chuyển mã C++ thành chương trình chạy được.

Chuẩn ngôn ngữ

Ưu tiên C++17 trở lên để code ngắn gọn, an toàn và dễ bảo trì hơn.

Chương trình C++ đầu tiên

// Chương trình in ra câu chào
#include <iostream>

int main() {
    // std::cout dùng để xuất dữ liệu
    std::cout << "Hello, C++!\n";

    return 0;
}

Giải thích từng dòng lệnh

  • #include <iostream>: Thư viện cho luồng vào/ra chuẩn (cin/cout).
  • int main(): Hàm chính, điểm bắt đầu thực thi.
  • std::cout << ...: Toán tử << đưa dữ liệu ra màn hình.
  • \n: Ký tự xuống dòng.
  • return 0;: Kết thúc chương trình thành công.
  • Dấu ;: Mỗi câu lệnh kết thúc bằng chấm phẩy.

Kiểu dữ liệu, Biến và Hằng

Kiểu Dữ Liệu

Xác định loại giá trị và bộ nhớ.

  • int: Số nguyên
  • double: Số thực độ chính xác cao
  • char: Ký tự đơn
  • bool: true/false

Bien (Variables)

Vùng nhớ lưu trữ dữ liệu, có thể thay đổi giá trị.

Cú pháp:
type tenBien = giaTri;

Ví dụ:
int age = 20;
double gpa = 3.25;

Hằng (Constants)

Giá trị cố định, không được thay đổi khi chạy.

Từ khóa const:
const double PI = 3.14159;

Ưu tiên: Dùng const thay vì #define để an toàn kiểu dữ liệu.

Sử dụng để thực hiện các phép tính toán học cơ bản.

Ký hiệu Ý nghĩa

Các Phép Toán: 2. Phép toán so sánh

Dùng để so sánh 2 giá trị. Trong C++, kết quả là kiểu bool (true hoặc false).

Ký hiệu Ý nghĩa
==Bằng nhau
!=Khác nhau
>Lớn hơn
<Nhỏ hơn
>=Lớn hơn hoặc bằng
<=Nhỏ hơn hoặc bằng
 
int a = 5, b = 8;

std::cout << (a == b) << "\n"; // 0
std::cout << (a != b) << "\n"; // 1
std::cout << (a < b) << "\n";  // 1

// Có thể in bool rõ ràng hơn:
std::cout << std::boolalpha << (a < b); // true

Các Phép Toán: 4. Gán và Tăng/Giảm

Phép Gán (Assignment)

Dùng để gán giá trị từ bên phải sang biến bên trái.

=a = 5
+=a += 3 (tương đương a = a + 3)
-=a -= 2 (tương đương a = a - 2)
*=, /=Hoạt động tương tự

Phép Tăng / Giảm

Tăng hoặc giảm biến đi 1 đơn vị.

  • ++a hoặc a++: Tăng a lên 1.
  • --a hoặc a--: Giảm a đi 1.
int x = 10;
x++; // x = 11
++x; // x = 12
x--; // x = 11

Nhập/Xuất và Comment trong C++

#include <iostream>
#include <string>

int main() {
    int age;
    std::string name;

    // cout: xuất thông tin
    std::cout << "Nhập tên: ";
    std::getline(std::cin, name);

    std::cout << "Nhập tuổi: ";
    std::cin >> age;

    std::cout << "Ban " << name
              << ", " << age << " tuổi.\n";
    return 0;
}

Lưu ý quan trọng

  • cin dùng toán tử >> để nhập dữ liệu.
  • getline(cin, str) dùng khi cần đọc cả dòng có dấu cách.
  • Nên dùng std:: rõ ràng thay vì using namespace std; trong dự án lớn.
  • Comment 1 dòng: // nội dung
  • Comment nhiều dòng: /* noi dùng */

Cấu Trúc Điều Khiển

Lệnh rẽ nhánh (if, switch) và Vòng lặp (for, while)

Cấu trúc rẽ nhánh (If - Else)

1. Lệnh if cơ bản

Chỉ thực hiện nếu điều kiện đúng.

if (age >= 18) {
    std::cout << "Duoc phep lai xe.";
}

2. Lệnh if - else

Xử lý 2 trường hợp đúng/sai.

if (score >= 5.0) {
    std::cout << "Đậu";
} else {
    std::cout << "Rớt";
}

3. Lệnh if - else if - else

Kiểm tra nhiều điều kiện liên tiếp nhau.

double score = 7.5;

if (score >= 8.0) {
    std::cout << "Xếp loại: Giỏi";
}
else if (score >= 6.5) {
    std::cout << "Xếp loại: Khá";
}
else if (score >= 5.0) {
    std::cout << "Xếp loại: Trung bình";
}
else {
    std::cout << "Xếp loại: Yếu";
}

Câu lệnh Switch - Case

Phù hợp khi so sánh một biến với các giá trị rời rạc (int, char, enum).

  • case: Giá trị cụ thể cần kiểm tra.
  • break: Dừng lại sau khi xử lý xong case.
  • default: Chạy khi không trùng case nào.
int choice = 2;

switch (choice) {
    case 1:
        std::cout << "Bạn chọn Thêm mới";
        break;
    case 2:
        std::cout << "Bạn chọn Chỉnh sửa";
        break;
    case 3:
        std::cout << "Bạn chọn Xóa";
        break;
    default:
        std::cout << "Lựa chọn không hợp lệ";
}

Bài tập vận dụng: If và Switch

  • Bài 1: Nhập số nguyên n, kiểm tra chẵn/lẻ.
  • Bài 2: Nhập 3 số thực a, b, c và tìm số lớn nhất.
  • Bài 3: Giải phương trình bậc nhất ax + b = 0, xử lý đầy đủ trường hợp a = 0.
  • Bài 4: Tính tiền điện theo bậc thang.
  • Bài 5: Viết máy tính cầm tay bằng switch-case cho các phép +, -, *, /.

Vòng lặp For

Định nghĩa và Cú pháp

Dùng khi đã biết trước số lần lặp.

for (khoi_tao; dieu_kien; buoc_nhay) {
    // Khối lệnh lặp lại
}
  • Khởi tạo: Chạy 1 lần đầu tiên.
  • Điều kiện: Đúng thì lặp tiếp, sai thì dừng.
  • Bước nhảy: Chạy cuối mỗi vòng lặp.
// In các số từ 1 đến 5
for (int i = 1; i <= 5; i++) {
    std::cout << i << " ";
}

std::cout << "\n";

// Vòng lặp chay lui
for (int i = 5; i > 0; i--) {
    std::cout << i << " ";
}

Bài tập vận dụng: Vòng lặp For

  • Bài 1: Nhập n, tính tổng S = 1 + 2 + ... + n.
  • Bài 2: Nhập n, tính giai thừa n!.
  • Bài 3: In bảng cửu chương của N (2 đến 9).
  • Bài 4: Tìm và in tất cả ước số của n.
  • Bài 5: Vẽ hình chữ nhật bằng dấu * bằng vòng lặp lồng nhau.

Vòng lặp While và Do-While

Vòng lặp while

Kiểm tra điều kiện trước, thích hợp khi chưa biết số lần lặp.

int n = 5;
while (n > 0) {
    std::cout << n << " ";
    n--;
}

Vòng lặp do - while

Thực hiện ít nhất 1 lần rồi mới kiểm tra điều kiện.

int age;
do {
    std::cout << "Nhập tuổi (>0): ";
    std::cin >> age;
} while (age <= 0);

Lưu ý: Nhớ cập nhật biến điều kiện để tránh vòng lặp vô tận.

Bài tập vận dụng: While và Do-While

  • Bài 1: Dùng do-while yêu cầu nhập số từ 1 đến 10 đến khi hợp lệ.
  • Bài 2: Dùng while đếm số chữ số của một số nguyên dương.
  • Bài 3: Đảo ngược thứ tự chữ số của n.
  • Bài 4: Tìm UCLN của a và b bằng thuật toán Euclid.
  • Bài 5: Trò chơi đoán số random 1-100 với gợi ý lớn hơn/nhỏ hơn.

Hàm và Hướng Đối Tượng Cơ Bản

Tái sử dụng mã nguồn và tổ chức chương trình

Hàm có giá trị trả về (Return)

Hàm nhận tham số, xử lý và trả kết quả thông qua return.

  • Kiểu trả về: int, double, bool...
  • return: Gặp là hàm kết thúc ngay.
int tinhTong(int a, int b) {
    return a + b;
}

int main() {
    int tong = tinhTong(8, 9);
    std::cout << "Tổng là: " << tong;
    return 0;
}

Hàm không trả về giá trị (void)

Hàm chỉ thực hiện thao tác như in thông tin, không cần trả giá trị.

  • Dùng từ khóa void.
  • Có thể không cần return;.
void inLoiChao(const std::string& ten) {
    std::cout << "==================\n";
    std::cout << "Xin chào bạn " << ten << "!\n";
    std::cout << "==================\n";
}

int main() {
    inLoiChao("Sinh viên");
    return 0;
}

Class và Object trong C++

class là khuôn mẫu, object là đối tượng được tạo từ class đó.

  • private: Thuộc tính nội bộ, không truy cập trực tiếp từ ngoài.
  • public: Hàm và thuộc tính cho phép truy cập.
  • Constructor: Hàm khởi tạo có tên trùng tên class.
class SinhVien {
private:
    std::string ten;
    double diem;

public:
    SinhVien(std::string t, double d) : ten(t), diem(d) {}

    void hienThi() const {
        std::cout << ten << " - " << diem << "\n";
    }
};

Mảng, vector, con trỏ và danh sách liên kết

Mảng và Vector trong C++

Mảng tĩnh (array)

Kích thước cố định, truy cập nhanh qua chỉ số.

  • Khai báo: int arr[5];
  • Khởi tạo: int arr[3] = {10,20,30};

Vector (STL)

Kích thước động, tự mở rộng khi cần.

  • #include <vector>
  • Có các hàm: push_back, size, at
#include <vector>

std::vector<int> diem = {7, 8, 9};
diem.push_back(10);

for (size_t i = 0; i < diem.size(); i++) {
    std::cout << diem[i] << " ";
}

Bài tập vận dụng:
1. Nhập N phần tử vào vector, tính tổng.
2. Tìm MAX/MIN trong vector.
3. Sắp xếp tăng dần bằng std::sort.

Struct và Class trong C++

struct và class đều có thể chứa thuộc tính và phương thức.

Khác biệt chính: mặc định struct là public, class là private.

struct ToaDo {
    double x;
    double y;
};

int main() {
    ToaDo p1{2.5, 4.0};
    std::cout << "x = " << p1.x
              << ", y = " << p1.y;
    return 0;
}

Bài tập vận dụng:
1. Tạo struct NhanVien, nhập mảng nhân viên và tìm lương cao nhất.
2. Tạo class PhanSo với hàm rút gọn và cộng 2 phân số.

Con trỏ và Tham chiếu (References)

Con trỏ lưu địa chỉ bộ nhớ, tham chiếu là bí danh cho biến đã tồn tại.

  • &a: Lấy địa chỉ của a.
  • *p: Truy cập giá trị mà p đang trỏ tới.
  • int& ref = a;: ref là tên khác của a.
int a = 10;
int* p = &a;
int& ref = a;

*p = 20;  // a = 20
ref = 30; // a = 30

std::cout << a;

Danh sách liên kết trong C++

Có 2 hướng: tự xây dựng Node bằng con trỏ hoặc dùng std::list trong STL.

[Head] -> [ Data: 10 | Next ] -> [ Data: 20 | Next ] -> NULL

Tự xây dựng Node

  • Mỗi node gồm dữ liệu và con trỏ next.
  • Cần quản lý bộ nhớ cẩn thận (new/delete).
struct Node {
    int data;
    Node* next;
};

Cài đặt danh sách liên kết cơ bản

Ví dụ lien ket 3 node theo cach don gian để hiểu cơ chế.

Bài tập nâng cao:
1. Viết hàm Node* createNode(int x) sử dụng new.
2. Viết hàm duyệt danh sách để in từng node.
3. Viết hàm them node vao đầu danh sách.

int main() {
    Node n1{10, nullptr};
    Node n2{20, nullptr};
    Node n3{30, nullptr};

    n1.next = &n2;
    n2.next = &n3;

    std::cout << "Node 2 là: "
              << n1.next->data << "\n";

    return 0;
}
